## diffview.vim(WIP)

A vim9 plugin inspired by nvim's diffview.nvim

## Installation

### vim-plug

1. Add the following line to your `~/.vimrc`:

```vim
call plug#begin()
...
Plug 'zeminzhou/diffview.vim'
...
call plug#end()
```

2. Run `:PlugInstall`.

## Start

You can use `:call ToggleDiffView()` to toggle this plugin.
